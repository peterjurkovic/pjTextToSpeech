pjToast notifications

pjToast is a simple Angular module for toast notifications. This Toast notification is inspired by ngToast with following significant differences:

  • The notification message is positioned absolute (to the center of screen)
  • Only one message is shown in given time (other additional messages are in the queue)

    app.controller('ToastController', function(Toast){
        Toast.success('This is success Toast message!');
    });


    app.controller('ToastController', function(Toast){
        Toast.info('This is info Toast message!');
    });


    app.controller('ToastController', function(Toast){
        Toast.warning('This is warning Toast message!');
    });


    app.controller('ToastController', function(Toast){
        Toast.danger('This is danger Toast message!');
    });
